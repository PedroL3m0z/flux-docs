A Flux transforma uma ou mais contas do Telegram em **instâncias** que você controla por HTTP — autentique, leia e envie mensagens, e receba cada atualização como eventos em tempo real ou webhooks assinados. Este guia percorre tudo o que dá para fazer, recurso por recurso.

## O que dá para fazer

- **Rode várias contas** — cada número do Telegram é uma [instância](/flux-docs/pt-br/instances/) com sua própria sessão, operada de forma independente num só gateway.
- **Leia e envie mensagens** — liste chats, navegue pelo histórico, envie texto e mídia (fotos, vídeos, documentos até 50 MB) por HTTP simples.
- **Reaja em tempo real** — novas mensagens, edições, exclusões, recibos de leitura, reações e mudanças de status via [SSE](/flux-docs/pt-br/events/) ao vivo, sem polling.
- **Receba webhooks assinados** — assine seu endpoint em qualquer subconjunto de eventos; entregas com assinatura HMAC, guardadas num outbox no Postgres e [com retentativa e backoff](/flux-docs/pt-br/webhooks/).
- **Controle o acesso** — [contas](/flux-docs/pt-br/accounts/) do dashboard com papéis admin / operator / viewer, senhas Argon2id, sessões criptografadas e URLs de webhook protegidas contra SSRF.

## No ar com um comando

A imagem única publicada junta API, PostgreSQL e Redis — sem setup, sem config:

```bash
docker run -d -p 3000:3000 -v flux_data:/data pedrooaj/flux-api
```

Depois abra o dashboard em `http://localhost:3000/dashboard`, ou as docs
interativas da API em `/docs`. Construída sobre **NestJS 11 + Prisma 7 (PostgreSQL) +
Redis**, com dashboard em **Vue 3**. Passo a passo completo em [Primeiros passos](/flux-docs/pt-br/getting-started/);
o fluxo da requisição até o endpoint está em [Como funciona](/flux-docs/pt-br/how-it-works/).

Prefere código? As mesmas ações estão a uma chamada HTTP de distância:

```bash
# Enviar uma mensagem por uma instância autorizada
curl -X POST \
  http://localhost:3000/telegram/instances/<id>/chats/<chatId>/messages \
  -H 'Authorization: Bearer <JWT>' -H 'x-api-key: <API_KEY>' \
  -H 'Content-Type: application/json' \
  -d '{"text":"Olá da Flux"}'
```

Navegue por toda a superfície na [Referência da API](/flux-docs/pt-br/api-reference/).
